# 2026-06-12 — правки выдачи задач, стартового экрана и защита

- На стартовом экране входа добавлена кнопка «?» для отправки сообщения об ошибке.
- В верхней панели рабочих ролей вместо имени показывается нейтральный заголовок «Мои задачи».
- Исправлена AJAX-обработка кнопок «Поменять исполнителя» и «Снять задачу»: сервер отдаёт JSON, фронтенд больше не зависает на HTML-ответах/редиректах.
- Во вкладке «Выданные сотрудникам» закреплены ширины колонок и кнопка «Снять» больше не уезжает за пределы таблицы.
- В `/assignments/report` выровнено отображение номера квартиры/помещения через отдельную колонку и адаптивный бейдж.
- Усилены серверные проверки прав: прямые ссылки на чужие объекты, задачи, отчёты, пользователей и конфликты синхронизации блокируются на сервере.
- Добавлена серверная CAPTCHA при входе, rate limit, блокировка аккаунта после серии неудачных входов, защита от session fixation и отзыв старых сессий после смены пароля.
- Добавлены безопасные cookie/headers, CSP, запрет iframe/clickjacking, nosniff, no-store для приватных страниц и HSTS при HTTPS.
- Добавлена allowlist-проверка загружаемых `.xlsx/.docx/.doc` по расширению, размеру и сигнатуре файла.
- Пароли больше не хранятся в открытом виде: поле `password_plain` очищается при запуске приложения.
- Локальный запуск больше не включает debug-режим по умолчанию; его можно включить только через `FLASK_DEBUG=1`.


## DOC upload fallback
- Разрешена загрузка старых `.doc` без обязательного LibreOffice: система сначала пытается конвертацию через LibreOffice/Microsoft Word/Pandoc/текстовый fallback, а затем использует встроенный шаблон, если конвертеров нет.
- Форма обновлена: `.doc` и `.docx` отображаются как поддерживаемые форматы.
